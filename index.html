HTML Forms Cheatsheat

Form Elements

Elements Syntax Description
The Input Element <input> The input tag specifies an input field where a user can enter data.
The Select Element <select> The select tag is used to make a drop down list.
The Textarea Element <textarea> Used in forms to collect user input like comments or reviews.
The Button Element <button> Defines a clickable button.
The Fieldset Element. <fieldset> The fieldset element is used to group related elements together.
