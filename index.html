HTML Cheatsheat

Form Elements

Elements Syntax Description
The Input Element <input> The input tag specifies an input field where a user can enter data.
The Select Element <select> The select tag is used to make a drop down list.
The Textarea Element <textarea> Used in forms to collect user input like comments or reviews.
The Button Element <button> Defines a clickable button.
The Fieldset Element. <fieldset> The fieldset element is used to group related elements together.

Common HTML Tags and their Use

Tags Syntax Description
The Comment Tag <--...--> The comment tag is used to add comments and description to your html documents. They are not displayed on the webpage. Their main purpose is increase readibility of your code.
The <a> Tag <a> The <a> defines hyperlinks, which is used to connect to other web pages. It's most important attribute is href which indicates the links description.
The Abbreviation Tag <abbr> The abbreviation tag defines an abbreviation or an acronym like HTML, CSS etc.
The Address Tag <address> The address tag defines the contact information of the author or owner of the document.
The Article Tag <article> The article tag specifies independent, self-contained content.An article should make sense on its own and it should be possible to distribute it independently from the rest of the site.
The Aside Tag <aside> The aside tag defines some content aside from the content it is placed in. The aside tag should indirectly be connected to the surrounding content.
The Blockquote Tag >blockquote> The blockquote specifies a section that is quoted form another source. Browsers usually indent blockquote.
The Body Element <body> The body element is almost used in all HTML projects. It defines the document's body.
